tomcat虚拟目录

1、tomcat-9.0.1/conf/server.xml 配置： path:指定访问该Web应用的URL入口，如：http://127.0.0.1:8080/helloApp1。 docBase:指定Web应用的文件路径，可以给定绝对路径，也可以给定相对于的appBase属性的相对路径，如果Web应用采用开放目录结构，则指定Web应用的根目录，如果Web应用是个war文件，则指定war文件的路径。 reloadable:如果这个属性设为true，tomcat服务器在运行状态下会监视在WEB-INF/classes和WEB-INF/lib目录下class文件的改动，如果监测到有class文件被更新的，服务器会自动重新加载Web应用。 2、idea使用虚拟目录：http://blog.csdn.net/cheney550995353/article/details/70185282
